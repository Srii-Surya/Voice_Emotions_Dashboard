# Voice Emotion Detection Dashboard

This project analyzes emotions from speech using speech-to-text and NLP techniques.
Built using Streamlit, SpeechRecognition, TextBlob, and Transformers.

## Features
- Upload audio files (.wav)
- Speech-to-text conversion
- Emotion detection from spoken text
- Visual emotion distribution

## Tech Stack
- Python
- Streamlit
- NLP (TextBlob, Transformers)
- Speech Recognition

## How to Run
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m streamlit run app.py
